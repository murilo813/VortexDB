# BUFFER MANAGER 
sem ele é feito: buffer = self.page_manager.load_page(page_id)
isso sempre: abre arquivo, le do disco, cria um bytearray, fecha arquivo
ISSO É EXTREMAMENTE CARO E PESADO

O buffer manager vai resolver isso guardando em memória
- A página só é lida do disco se não estiver no buffer.

dentro desse butter ele vai guardar:
self.pages = {
   page_id: {
      "buffer": bytearray, # pagina
      "dirty": False, # foi alterada?
      "pin": 0 # quantos usuarios estao usando essa pagina? (pin > 0 NAO PODE REMOVER DA MEMORIA)
   }
}
